# Hospital Management System (C++)

## Description
This is an **Advanced Hospital Management System** developed in C++ that allows managing patients, appointments, billing, and admin login with **file persistence**.  
It includes features like **deposit/withdraw (billing)**, **transaction history**, and **deleting patient records**. All data is stored in text files for persistence.

---

## Features
- Create new patient accounts
- Login to patient account
- Manage appointments and bills
- View patient details
- View transaction history
- Delete patient account
- File-based storage (`patients.txt`, `appointments.txt`, `bills.txt`)
- Simple and interactive console interface
